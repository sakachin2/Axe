-Continuation of InsertLine.

             OPT INS_LINE [on | off ]

             on/off  :After Inn edit line cmd, Enter key insert a null line automaticaly.
                      Default is on,
                      Autoatic insertion stops under the condition of
                      cursor is not on the last inserted line,
                      or last inserted is left as null line,
                      or another line cmd exist,
                      or there is cmd on the cmd input line,
                      or line deletion is done.
