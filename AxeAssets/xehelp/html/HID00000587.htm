(a)dtm
      start xe on dtterm with setup X resource.
      0001 notify X resource file.
      0002 -name parameter specify application name.
           Application name is not used for X resource name for dtterm.
           So,use DtTerm application class name on X resource file.

        0001|export XENVIRONMENT=$HOME/.Xdefault.xe
        0002|dtterm -name dtterm_xe &
