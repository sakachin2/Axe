(2). xeaix___.tgz contains following files.
     (See xereade.me for not described bellow.)

      . xeaixe.doc   Doc for AIX version(this file)
      . xeaixj.doc   Doc for AIX version(Japanese:SJIS)
      . xelnxe.doc   Doc for Linux version(this file)
      . xelnxj.doc   Doc for Linux version(Japanese:SJIS)

      . aixbin.tar   tar of binary
        . xekbchk    tool for keyboard chk.

        see xereade.me for other binary.

      Followings are in sample directory.

      . teraterm.ini Sample TeraTerm setup file I use.
      . xett106a.cnf Sample TeraTerm keyboard setup file
                     for IBM 106 keyboard.

      . xelt         Start shell for Linux telnet user.
                     It is for RedHat v9,but it may work for other
                     distribution.
      . xeaixkey.rh9 Translation file used in xelt.

      . atm          Sample shell to start aixterm for xe.
      . dtm          Sample shell to start dtterm for xe.
      . .Xdefaults.xe  Sample .Xdefaults file to use xe
                       on dtterm or aixterm.

      . prl,prl2,prlw,prp,prpw,prpages
                   Sample shell script to use xprint,each are for
                   landscape,2 column landscape,small font landscape,
                   portrate,small font portrate(Those are for
                   PostScript) and for PAGES printer.
        (Note)To use these shell,specify as following in xe.ini file.
              ex) XPrint_Command_LNX="prpages" #("")# XP command
