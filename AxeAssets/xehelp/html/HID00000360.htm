= . command
        .set label to the line to be used by Find,Sort etc. command to limit
         the line to apply the command.
        .specify alpha-numeric after ".".
        .It can be used as line mark. Set any label on the line, search those lines by "f *e" cmd.


        .ex) sort line 9 to 14
