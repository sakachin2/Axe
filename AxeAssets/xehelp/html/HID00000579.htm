(b) Default Esc string translation of xe.
        (Terminfo defined string is accepted under fullscreen console mode)

        Note. "^[" is  notation of 0x1b.
              "*SCO"(^[[M-->^[[Z,^[[a-->^[[v) means for PuTTY keyboard type=SCO.
              Use --NOSCO option to inactivate SCO.
              And SCO assigns ^[Z to S+F2, it is duplicated to previous assign ment to S+Tab.
              Use --Z2 option to use ^[Z to S+F2.
